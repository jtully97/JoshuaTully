Creating Objects

In javascript an object has properties associated to it, those properties in simple terms are variables attatched to the object.

Lets try and relate this to a real life example. Imagine a car, think of all the characteristics about that car, evrything from make, model, color, year etc. those are properties related to that car, and we can call those the cars variables.

Now when writting code involving the creation of several cars, it can become extremely redundant attempting to dynamically create a car each and every time you want to add a car or alter the cars properties. This is where the creation of a car object helps drastically.
