Advanced Array Methods

Here we will discuss a few advanced array methods.

pop

The pop() method removes the last element contained in an array.

The pop()returns the last value of an array as well so cars.pop() returns cruise.

push

The push() adds a element to the end of an array. The push() in itself returns the new length of the array.

shift

The shift() method removes the first element of an array and moves all remaining elements to the left. This is equivalent to popping but instead working on the first element.

The shift() returns the element that was removed.

unshift

The unshift() adds an element to the beginning of an array. The unshift() method returns the new array length.

forEach

The forEach() calls a function once for each array element.

findIndex

The findIndex() method returns the first index of the first element in an array to satisfy the function.

filter

The filter() method creates a new array with elements from another array that pass a test function.

find

The find() method finds the first element in an array that passes a test then returns the value it found.
