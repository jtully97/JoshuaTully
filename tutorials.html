Unary Operators

In terms of simplicity, the most simple of operators in JavaScript are unary operators. A unary operator works on one operand.
The unary operators in JavaScript are:

  • Unary plus(+) - convert an operand into a number.
  • Unary minus(-) - convert an operand into a number and negate the following.
  • Increments (++) - add one to operand
  • Decrement (--) - subtract one from operand
  • Logical Not (!) - Converts to boolean value then negates it
  • Bitwise not (~) - Inverts all the bits in the operand and returns a number
  • typeof - comes before the operand, this returns a string of the data type of the operand.
  • delete - Deletes specific index of an array or specific property of an object
  • Void operator - Discards a return value of an expression

Logical not

This operator is placed before the operand. it converts the operand into its boolean equivalent before negating it.

Bitwise not

Performs a bianary NOT operation, this operator inverts all the bits in the operand and then returns a number.

typeof

This operator comes before the operand. it returns a string indicating the data type of the operand.

delete

This operator is placed before the operand. it deletes the values of a specified index of an array and a specific property of a given object.

Void operator

It precedes an operation. It discards the return value of an expression, meaning it evaluates an expression but then returns undefined.

The void operator is useful to specify an expression as a hypertext link. The expression is evaluated but does not loaded in place of the current document.
